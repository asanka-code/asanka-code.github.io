Tools and Resources

The following are various resources and tools that I found useful for various purposes.

Finding and Accessing Information

  • Reading Paywalled Content:
    Whenever you come across a web page (such as news articles and blogs) which are placed behind a paywall, the following tools can be useful in accessing an unpaywalled copy of the content.

    • Paywall Reader: https://paywallreader.com
    • Internet Archive: https://web.archive.org
    • Library Genesis: https://libgen.rs
    • PDF Drive: https://www.pdfdrive.com

  • Alternative Web Search Engines:
    Avoiding Big-Tech, these search engines can be used to search the web in a privacy preserving manner.

    • DuckDuckGo: https://duckduckgo.com
    • Brave Search: https://search.brave.com
    • Mojeek: https://www.mojeek.com
    • Stract: https://stract.com
    • SearXNG: https://searxng.online (all instances: https://searx.space)

  • Literature Survey Tools:
    The following tools are useful in conducting literature surveys.

    • Google Scholar: https://scholar.google.com
    • OpenAlex: https://openalex.org
    • Scholar in Internet Archive: https://scholar.archive.org
    • Trending Papers: https://trendingpapers.com
    • Connected Papers: https://www.connectedpapers.com
    • arXiv: https://arxiv.org
    • ResearchGate: https://www.researchgate.net
    • Publish or Perish: https://harzing.com/resources/publish-or-perish
    • NotebookLM: https://notebooklm.google.com
    • DeepSeek: https://chat.deepseek.com
    • NaturalReaders: https://www.naturalreaders.com/online
