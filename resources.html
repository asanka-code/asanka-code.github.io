Resources

The following are various resources and tools that I found useful for various purposes.

  • Reading Paywalled Content:
    Whenever you come across a web page (such as news articles and blogs) which are placed behind a paywall, the following tools can be useful in accessing an unpaywalled copy of the content.

    • Paywall Reader: https://paywallreader.com
    • Internet Archive: https://web.archive.org

  • Literature Survey Tools:
    The following tools are useful in conducting literature surveys.

    • Google Scholar: https://scholar.google.com
    • OpenAlex: https://openalex.org
    • Trending Papers: https://trendingpapers.com
    • Connected Papers: https://www.connectedpapers.com
    • arXiv: https://arxiv.org
    • ResearchGate: https://www.researchgate.net
    • Publish or Perish: https://harzing.com/resources/publish-or-perish

  • Important Conferences and Journals:
    The following are some of the important conferences and journals related to cybersecurity and digital forensics domains that I am interested in publishing.

    • Forensic Science International: Digital Investigation (FSI:DI): https://www.sciencedirect.com/journal/forensic-science-international-digital-investigation
    • DFRWS (USA/EU/APAC): https://dfrws.org
    • IEEE Symposium on Security and Privacy (S&P): https://www.ieee-security.org
    • USENIX Security Symposium: https://www.usenix.org/conference/usenixsecurity24
    • IEEE Transactions on Information Forensics and Security: https://dl.acm.org/journal/tifs
    • ACM Computer and Communications Security (CCS) Conference: https://dl.acm.org/conference/ccs
    • The International Conference on Availability, Reliability and Security (ARES): https://www.ares-conference.eu

  • Miscellaneous:
    Various tools for useful and fun activities.

    • Spurious Scholar: https://tylervigen.com/spurious-scholar
    • Scholar Goggler: https://shiny.rcg.sfu.ca/u/rdmorin/scholargoggler7
    • PHD Comics: https://phdcomics.com
    • ErrantScience: https://errantscience.com
